## 앱의 기능
1. 장보기 체크리스트
2. 냉장고 재고확인 기능
3. 유통기한 임박한 상품 알림 서비스 (Firebase FCM)
4. 영수증 인식기능 (네이버 제공 API)
5. (식재료 기반 레시피 추천 기능)
6. (직거래 마켓 연동) 

## 앱의 목표
> 불필요한 음식물 쓰레기를 줄여 환경에 긍정적 영향
